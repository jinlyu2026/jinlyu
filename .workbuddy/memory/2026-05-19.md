# 2026-05-19 工作日志

## 创建心理咨询技术单词连连看游戏

为应用心理学硕士项目讲师创建了单词连连看游戏，帮助学生快速记忆专业术语。

### 游戏特点
**模式一：单人模式**
- 3个难度等级：基础术语、咨询技术、高级概念
- 每个等级8对单词（中英文配对）
- 计时器和得分系统
- 卡片翻转动画效果

**模式二：团队竞赛**
- 支持2-6个团队同时竞赛
- 轮流答题机制
- 实时显示各团队得分
- 自动判断获胜团队并显示庆贺画面

### 单词内容
- **基础术语**（8对）：共情、倾听、信任、尊重、真诚、接纳、理解、支持
- **咨询技术**（8对）：积极倾听、开放式提问、反射技巧、面质、解释、自我表露、即时性、沉默
- **高级概念**（8对）：移情、反移情、阻抗、无条件积极关注、一致性、同理心疲劳、治疗联盟、自我觉察

### 技术实现
- 单页HTML应用，无需服务器
- 响应式网格布局，适合课堂投影
- 洗牌算法确保每次游戏卡片位置随机
- 支持中英文双语界面
